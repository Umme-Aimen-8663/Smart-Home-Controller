# Smart Home Controller - Flutter

**Task 6: Home Automation App**  
Flutter Fellowship 2026

## Features
- MQTT-based Smart Home Device Control
- Real-time device toggling (Light, Fan, AC, Plug)
- Provider State Management
- Responsive Device Cards
- Dark/Light Theme Support
- Connection to Public MQTT Broker

## Tech Stack
- Flutter + Dart
- `mqtt_client` package
- Provider (State Management)
- Shared Preferences (planned)

## How to Test
1. Run the app
2. Click **"Connect to MQTT"**
3. Toggle devices ON/OFF
4. (Optional) Use MQTTX desktop app and subscribe to `home/devices/#` to see commands

**Note**: Due to network issues with public brokers, demo mode is enabled for reliable demonstration.
